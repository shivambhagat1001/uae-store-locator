# UAE Store Locator

A store locator web app built using OpenStreetMap (Nominatim API).

## 🚀 Features
- Location autocomplete (UAE restricted 🇦🇪)
- Latitude & Longitude extraction
- Distance calculation using Haversine formula
- Nearest store sorting
- Input validation (must select from suggestions)
- Duplicate location filtering

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenStreetMap API (Nominatim)
